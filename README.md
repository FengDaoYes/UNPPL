# UNPPL
Windows进程PPL属性研究的辅助工具，仅用于合法授权环境下的技术学习与研究。

## 操作步骤
1. 安装驱动：`UNPPLControl.exe install`
2. 取消指定进程的PPL属性：`UNPPLControl.exe use 720`
3. 卸载驱动：`UNPPLControl.exe uninstall`
